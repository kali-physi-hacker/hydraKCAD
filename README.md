# HydraKCAD
This project is a CAD Kernel. The aim of this project is for educational purposes; to see mathematical and physics concepts applicable to computer graphics such calculus, linear algebra, light, etc. come to life.
This project in no way intends to replace existing robust CAD Kernels.   
**C++** is the programming language of choice, overtime plugins and other features may be implemented in **Rust**.

### Technical Requirements
- Programing Language: C++
- Compiler: Any C++ compile (GNU, etc) that supports the C++20 language standard.
